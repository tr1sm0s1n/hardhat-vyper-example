Clone the repository

```bash
git clone https://github.com/tr1sm0s1n/hardhat-vyper-example
cd hardhat-vyper-example
```

Install dependencies

```bash
pnpm install
```

Test the contract

```bash
pnpm test
```

Deploy the contract (edit hardhat.config.cjs at first)

```bash
pnpm run deploy
```
